# EnergyTrack Project

## Goal
Energy consumption monitoring REST API + Flutter mobile app.
Built for IQB Solutions job application portfolio.

## Tech Stack
- Backend: ASP.NET Core Web API (.NET 10)
- Mobile: Flutter
- Database: PostgreSQL (running in Docker)
- Auth: JWT Bearer Token
- Container: Docker + docker-compose

## Project Structure
- /backend → ASP.NET Core API
- /mobile  → Flutter app

## API Conventions
- Base URL: /api/v1/
- Always include error handling
- Always include XML comments on public methods
- Use Repository pattern
